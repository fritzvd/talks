## Fun facts
* 1.79 MHz 6502-like CPU
* 5.37 MHz PPU
* 2 kb RAM
* 256 x 240 (NTSC 256x224)
* 64 colors, with duplicates (i.e. 54)
* 3 colors per sprite / tile
* 8 'keys' (left, right, etc)
* 8 sprites per scanline
* CHR and PRG ROM
